Class Fighter

Clase abstracta Fighter que define los atributos y funciones comunes para las subclases de los universos de los luchadores

Se define 8 atributos: dos tipo string, name y catching_phrase; y 6 de tipo number height, weight, attack, defense, speed y healt. Cada atributo se autodefinen.
