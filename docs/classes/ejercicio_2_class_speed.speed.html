Class Speed

Clase Speed que sirve para realizar las conversiones de la magnitud de velocidad

Se define 2 atributos, valor de tipo number que representa el valor númerico y unidad1 de tipo string que define la en que unidad se encuentra.

Se tienen 5 métodos, getValor, getUnidad1, setValor, setUnidad1 y conversion.

Se ha implementado para que se haga las siguientes conversiones: -kilometro por hora -> metro por segundo -kilometro por hora -> milla por hora -metro por segundo -> kilometro por hora -milla por hora -> kilometro por hora
